# Basic Examples

Learn Anaphora through practical examples. Each example walks you through creating a complete job from start to finish.

## Available Examples

| Example | Description |
|---------|-------------|
| [Kibana Dashboard Report](./basic-examples/kibana-dashboard-report) | Schedule a daily PDF report of your Kibana dashboard |
| [Kibana Alert](./basic-examples/kibana-alert) | Send notifications when conditions are met |
| [Conditional Report](./basic-examples/conditional-report) | Reports that only send when criteria match |
| [Grafana Dashboard Report](./basic-examples/grafana-dashboard-report) | Capture and deliver Grafana dashboards |

## Prerequisites

Before starting these examples, ensure you have:

- Anaphora installed and running
- A connection configured to your Kibana or Grafana instance
- At least one delivery interface set up (email, Slack, etc.)

Start with the [Kibana Dashboard Report](./basic-examples/kibana-dashboard-report) example for a complete walkthrough.
