# Delivery Interfaces

Delivery Interfaces define how and where reports are sent after capture and composition.

## Overview

Anaphora supports multiple delivery channels to ensure your reports reach the right people through their preferred communication tools.

## Supported Interfaces

| Interface | Description | Status |
|-----------|-------------|--------|
| Email | SMTP-based email delivery | Available |
| Slack | Slack workspace integration | Available |
| MS Teams | Microsoft Teams channels | Available |
| Webhook | Custom HTTP endpoints | Available |

## Configuration

Delivery interfaces are configured globally in **Settings** and then selected per-job in the **Deliver** tab.
