# Spaces

Spaces provide multi-tenant workspace isolation in Anaphora.

## Overview

Spaces allow you to:
- Separate jobs and reports by team or project
- Apply different permissions per space
- Isolate data between tenants
- Manage resources independently

## Creating Spaces

1. Navigate to **Settings** > **Spaces**
2. Click **Create Space**
3. Enter name and description
4. Configure space settings
5. Save

## Space Configuration

| Setting | Description |
|---------|-------------|
| Name | Space identifier |
| Description | Purpose of the space |
| Default | Set as default for new users |
| Retention | Override global retention policy |

## User Assignment

Assign users to spaces with specific roles:

1. Open space settings
2. Go to **Members** tab
3. Add users and assign roles
4. Users only see jobs in their spaces

## Switching Spaces

Users with access to multiple spaces can switch via the space selector in the navigation bar.

## Resource Isolation

Each space has independent:
- Jobs and schedules
- Reports and history
- Delivery configurations
- Templates

## Best Practices

- Create spaces by team or department
- Use descriptive names
- Assign minimal required permissions
- Review space membership regularly
